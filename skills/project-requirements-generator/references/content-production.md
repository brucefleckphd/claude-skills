## 8. DECISION RULES

### Content Decisions
| Situation | Decision |
|-----------|----------|
| Topic overlap with existing content | Link to existing, add new angle |
| Controversial take | Support with evidence, acknowledge other views |
| Technical complexity | Simplify first, offer "deep dive" as optional |
| Length running long | Split into series rather than cutting depth |

### Style Decisions
| Situation | Decision |
|-----------|----------|
| Active vs. passive voice | Active default |
| Jargon | Define on first use, minimize overall |
| Examples | Prioritize real over hypothetical |
| CTAs | One primary CTA, soft secondary max |
