# Build the Rundown

Create production-ready run-of-show documents. This skill produces minute-by-minute schedules that keep events running smoothly.

## When to Use This Skill

- Stage shows and presentations
- Conference sessions
- Live broadcasts
- Workshops with multiple segments
- Any timed event production

## Required Inputs

- Event overview and duration
- All segments/activities planned
- Speakers/participants
- Technical requirements
- Venue/platform constraints

## Output Format

```markdown
# Run of Show: [Event Name]

**Date:** [Date]
**Venue/Platform:** [Location]
**Total Duration:** [Length]
**Production Contact:** [Name/Phone]
```
