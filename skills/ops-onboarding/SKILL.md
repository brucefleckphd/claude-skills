# Design Onboarding Sequence

Set new team members and clients up for success from day one. This skill creates comprehensive onboarding experiences.

## When to Use This Skill

- New employee onboarding
- Client onboarding
- Contractor kickoff
- Community member welcome
- Product user onboarding

## Required Inputs

- Onboarding type (employee/client/member)
- Duration of onboarding period
- Key milestones to achieve
- Systems and tools to learn
- Team members involved
