# Plan an Event

Transform event ideas into executable plans. This skill creates comprehensive event blueprints with timelines, budgets, and logistics.

## When to Use This Skill

- Conference or summit planning
- Workshop or training events
- Community meetups
- Client events
- Virtual events and webinars
- Launch parties

## Required Inputs

- Event type and purpose
- Target audience and expected attendance
- Date parameters (specific or range)
- Budget range
- Key goals/success metrics
- Location preferences (in-person/virtual/hybrid)
