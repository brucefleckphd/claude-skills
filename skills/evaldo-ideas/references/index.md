# Evaldo Ideas Frameworks Index

Quick reference for all idea generation frameworks.

## Idea Generation Methods

| Framework | Description | Use When |
|-----------|-------------|----------|
| ideas-new-mechanism | Find new HOW for existing desire | Product needs fresh angle |
| ideas-major-developments | Track industry news/events | Creating timely promotions |
| ideas-stockholm-shoe | Study competitors for gaps | Entering competitive market |

## Idea Development

| Framework | Description | Use When |
|-----------|-------------|----------|
| ideas-idea-sources | Where to find raw material | Starting research phase |
| ideas-10-questions-brainstorm | Idea-to-copy bridge | Converting idea to structure |
| ideas-competitor-analysis | Systematic competitor study | Need competitive intelligence |

## Framework Relationships

```
ideas-idea-sources (gather raw material)
    ↓
ideas-new-mechanism OR ideas-major-developments OR ideas-stockholm-shoe
    ↓
philosophy-three-box (filter)
    ↓
ideas-10-questions-brainstorm (structure)
```

## Quick Idea Validation

Before developing any idea further:

1. Does it pass the Three Box Check?
   - [ ] Emotional (moves you)
   - [ ] Simple (child understands)
   - [ ] New (never seen before)

2. Does it fit the One Belief Formula?
   - [ ] New opportunity identified
   - [ ] Connects to true desire
   - [ ] Has unique mechanism
