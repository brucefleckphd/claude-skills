## How to Use Bundles

1. **Identify your task** from the bundle list above
2. **Load core frameworks** first - these are mandatory
3. **Add supporting frameworks** based on specific needs
4. **Work through frameworks in order** - they build on each other
5. **Reference specific steps** within each framework for execution
