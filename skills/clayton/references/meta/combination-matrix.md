# Framework Combination Matrix

> **Purpose**: Show which frameworks pair together for maximum impact. Use this to build complete campaigns, not just isolated tactics.

## How to Use This Matrix

1. **Start with your primary task** (headline, body copy, offer, etc.)
2. **Find the primary framework** for that task
3. **Add supporting frameworks** from the "Pairs With" column
4. **Layer psychology frameworks** for emotional depth

## Synergy Scores

**Triple-A Combinations** (Maximum synergy):
- 12-Step System + Copy Rhythm + Proof Stacking
- Offer Architecture + Value Stacking + Guarantee Power
- Pre-Event Planning + Grassroots Excitement + Post-Event Follow-Up
- Headline Templates + Direct Benefit Paradox + Curiosity Creation

- Any framework + Specificity Power (adds credibility to everything)
- Any framework + Sensory Language (adds vividness to everything)

## Anti-Patterns: Combinations to Avoid

| Don't Combine | With | Why |
|--------------|------|-----|
| Hard urgency | Soft relationship copy | Tone clash |
| Multiple emotion appeals | Same piece | Dilutes impact |
| Complex offer | Complex close | Overwhelms reader |
| Fear appeal | Exclusivity appeal | Mixed emotions |
