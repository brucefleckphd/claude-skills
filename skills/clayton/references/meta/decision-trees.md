## Decision Tree 2: "What Problem Am I Solving?"

```
START: What's not working?
│
├─► LOW OPEN RATES
│   └─► USE: Subject Line Templates
│       + Curiosity Creation
│       + Specificity Power
│
├─► LOW CLICK-THROUGH
│   └─► Check your:
│       ├─► Headlines → USE: Six Headline Maxims
│       ├─► Body copy → USE: Momentum Sequence
│       └─► CTAs → USE: Action Device Formula
│
├─► LOW CONVERSION (people read but don't buy)
│   └─► Missing element?
│       ├─► Proof → USE: Social Proof Stacking System
│       ├─► Trust → USE: Authority Building Formula
│       ├─► Value clarity → USE: Offer Clarity Formula
│       ├─► Risk → USE: Guarantee Power Formula
│       └─► Urgency → USE: Scarcity Timing Formula
│
├─► HIGH REFUNDS
│   └─► Problem area:
│       ├─► Overpromising → USE: Ethical Persuasion Guide
│       ├─► Wrong customers → USE: Qualification Copy
│       └─► Poor onboarding → USE: Confirmation Email Formula
│
├─► FLAT RESPONSE (no engagement)
│   └─► Try:
│       ├─► Story approach → USE: Story Architecture
│       ├─► Emotion approach → USE: Dominant Emotion System
│       └─► Controversy → USE: Contrarian Positioning
│
└─► CAN'T DIFFERENTIATE
    └─► USE: USP Development Framework
        + Mechanism Differentiation
        + Category Design Principles
```

## Decision Tree 4: "What's My Market Sophistication Level?"

```
START: How aware is your market?
│
├─► LEVEL 1: Unaware (don't know problem exists)
│   └─► Strategy: Educate first
│       USE: Story-based leads
│       + Problem revelation
│       + Indirect headlines
│
├─► LEVEL 2: Problem-aware (know problem, not solutions)
│   └─► Strategy: Agitate problem
│       USE: Problem-Pain Oscillation
│       + Future Pain Visualization
│       + Solution introduction
│
├─► LEVEL 3: Solution-aware (know solutions exist)
│   └─► Strategy: Differentiate your solution
│       USE: Mechanism Differentiation
│       + Unique Positioning
│       + Proof emphasis
│
├─► LEVEL 4: Product-aware (know your product)
│   └─► Strategy: Show why NOW
│       USE: Urgency/Scarcity systems
│       + Special offer emphasis
│       + Risk reversal
│
└─► LEVEL 5: Most aware (ready to buy)
    └─► Strategy: Remove friction
        USE: Order Form Optimization
        + Clear CTA
        + Fast checkout
```
