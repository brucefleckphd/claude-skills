# Evaldo Bullets

Evaldo Albuquerque's bullet (fascination) frameworks. Bullets are mini-headlines - each one creates desire and curiosity. Great bullets can sell by themselves.

## When to Use This Skill

- Writing bullets for offer section
- Creating fascinations
- VSL "what you'll discover" sections
- Email teaser bullets
- Any list of benefits

## Crown Jewels

1. **The Curiosity+Benefit Formula** - The core of every bullet
2. **The Parenthesis Technique** - Adding impact with parenthetical
3. **The Page Number Trick** - Creating tangibility
4. **The "What/Why/How" Stack** - Three bullets from one idea

## Complete Workflow

1. **List all product elements** - Every feature/benefit
2. **Find the curiosity angle** - What's intriguing?
3. **Add the benefit** - What do they get?
4. **Apply parenthesis** - Amplify impact
5. **Stack strongest first** - Order by power
6. **Test for open loops** - Does each create desire?

## Critical Evaldo Quote

> "Every bullet is a mini-headline. Curiosity plus benefit.
> The curiosity hooks them, the benefit sells them.
> A great bullet can sell the product by itself."
