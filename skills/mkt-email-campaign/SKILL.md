# Design Email Campaign

Build email campaigns that convert. This skill creates strategic email sequences with optimized copy, timing, and CTAs.

## When to Use This Skill

- Welcome/onboarding sequences
- Product launch campaigns
- Promotional offers
- Re-engagement campaigns
- Event promotion
- Newsletter planning

## Required Inputs

- Campaign goal and type
- Target audience segment
- Key offer or message
- Sequence length
- Send timing preferences

## Output Format

```markdown
# Email Campaign: [Campaign Name]

**Type:** [Welcome/Launch/Promo/Nurture]
**Audience:** [Segment]
**Goal:** [Primary objective]
**Sequence Length:** [X emails over Y days]
```
