│       │       + 4 C's + 2 H's Check

│           └─► USE: Dual Journey Structure
│               + Energy Audit
│               + Full Close System
│               + Story Integration
│
├─► HEADLINE
│   └─► What's the market sophistication?
│       ├─► Stage 1-2 (new market)
│       │   └─► USE: Direct benefit headlines
│       │       + Big Idea Framework
│       │
│       ├─► Stage 3-4 (competitive market)
│       │   └─► USE: Mechanism Headline
│       │       + Add/Subtract/Divide
│       │
│       └─► Stage 5 (saturated market)
│           └─► USE: Indirect/Curiosity approach
│               + Opposite Headline Technique
│               + Big Idea (must be NEW perspective)
│
├─► LEAD/OPENING
│   └─► What's the audience state?
│       ├─► Skeptical (tried everything)
│       │   └─► USE: Acknowledgment Lead
│       │       + Resistance-Free Lead
│       │
│       ├─► Unaware (don't know problem)
│       │   └─► USE: Story Lead
│       │       + Nightmare Story
│       │
│       └─► Pain-aware (know the problem)
│           └─► USE: Dual Journey Lead
│               + Future Pace

├─► BULLETS
│   └─► USE: Bullet Polishing Method
│       + Specificity Stack
│       + Mechanism Insertion

│       Best for: Weight loss, investing, self-improvement
│       Pairs with: Acknowledgment Lead

│       Pairs with: Future Pace

│   └─► USE: Resistance-Free Lead

│       Message: Come at it indirectly
│
├─► WANTS REVENGE (prove doubters wrong)
│   └─► USE: Payback/Revenge Trigger
│       Best for: Success products
│       Pairs with: Transformation Imperative
│       Message: "Show them they were wrong"
│
├─► WANTS TO BELONG (insider status)

## Decision Tree 4: "Market Sophistication Level?"

```
START: How aware is your market? (Schwartz 5 Stages)
│
├─► STAGE 1: You're first to market
│   └─► Strategy: Just state the benefit
│       USE: Direct headline
│       Example: "Lose weight"
│       No mechanism needed
│
├─► STAGE 2: Competitors have emerged
│   └─► Strategy: Make claim bigger
│       USE: Amplified benefit headline
│       Example: "Lose 20 pounds in 2 weeks"
│       Still claim-based
│
├─► STAGE 3: Claims are exhausted
│   └─► Strategy: Add mechanism
│       USE: Mechanism Headline
│       Example: "Lose weight while you sleep"
│       HOW becomes the differentiator
│
├─► STAGE 4: Mechanisms are exhausted
│   └─► Strategy: Better/newer mechanism
│       USE: Mechanism + specificity
│       Example: "Miracle herb from the Amazon"
│       Must be MORE compelling mechanism
│
└─► STAGE 5: Market is saturated
    └─► Strategy: Indirect approach, big idea
        USE: Big Idea Framework
            + Indirect Lead
            + Curiosity-based
        Example: "Why French women never get fat"
        NOT a claim - a NEW PERSPECTIVE
```

## Decision Tree 5: "Which Story Type Do I Need?"

```
START: What purpose does the story serve?
│
├─► INTRODUCING PRODUCT/COMPANY
│   └─► USE: Origin Story Formula
│       Pattern: Problem existed → Founder struggled →
│                Found solution → Refined it → Shares with you
│       Benefit: Shows value without bragging
│
├─► SHOWING CONSEQUENCES OF INACTION
│   └─► USE: Nightmare Story
│       Pattern: Paint the disaster without solution
│       Examples: Lost all money, health crisis, failure
│       Benefit: Activates fear, creates urgency
│
├─► DEMONSTRATING RESULTS
│   └─► USE: Transformation Story
│       Pattern: Before (suffering) → Discovery → After (thriving)
│       Benefit: Proof through narrative
│
├─► ADDING CREDIBILITY
│   └─► USE: Ally Appearance
│       Pattern: Hero meets mentor who opens eyes
│       Example: "I met a man who changed everything"
│
├─► NEED IMMEDIATE ATTENTION
│   └─► USE: Body on the Floor Opener
│       + Middle-of-Action Lead
│       Pattern: Start at peak drama, explain later
│
└─► STRUCTURE ANY STORY
    └─► USE: Hero's Journey Template
        1. Ordinary World
        2. Call to Adventure
        3. Obstacles and Enemies
        4. Allies appear
        5. Turning Point
        6. Return with Knowledge
```

## Decision Tree 6: "How Do I Evaluate This Copy?"

```
START: What aspect needs checking?
│
├─► OVERALL QUALITY
│   └─► USE: 4 C's + 2 H's Check
│       □ Clarity - Easy to read?
│       □ Compelling - Head AND heart?
│       □ Credible - Believable? Proof?
│       □ Change - Transforms life?
│       □ Honest - Would you say to Mom?
│       □ Heart-to-Heart - Your heart to theirs?
│
├─► DUAL JOURNEY CHECK
│   └─► At each section, ask:
│       □ Moving logic forward?
│       □ Moving emotion forward?
│       If neither - cut or rewrite
│
├─► ENERGY LEVEL
│   └─► USE: Energy Audit
│       1. Find best paragraph
│       2. Mark "dead" sections
│       3. Bring all up to best level
│       4. Use Read Aloud Test
│
├─► AUTHENTICITY
│   └─► USE: Just Talk Method checks
│       □ Sounds like conversation?
│       □ Could say across a table?
│       □ No "marketing speak"?
│       □ Dear Mom authenticity?
│
├─► SIMPLICITY
│   └─► USE: Caveman Simple Test
│       □ Distracted reader understands?
│       □ 12-year-old could get it?
│       □ No jargon or complexity?
│
└─► CLOSE STRENGTH
    └─► Check all elements:
        □ Future Pace present?
        □ Crossroads decision?
        □ Risk transferred?
        □ Scarcity real?
        □ Exact instructions given?
        □ Passion in delivery?
```
