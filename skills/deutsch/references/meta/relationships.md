## Category Families (Sibling Relationships)

### Psychology (14 frameworks)
- **Tier 1**: 10 Emotional Triggers, Fear Beats Greed, Transformation Imperative
- **Tier 2-3**: Touch of Larceny, Freedom Framework, Revenge Motivator, etc.
- **Relationship**: All psychology frameworks support other categories

### Strategy (12 frameworks)
- **Tier 1**: 4 C's, 2 H's, Big Idea, Just Talk Method, Market Sophistication
- **Tier 2-3**: Deep Dig, Prospect Immersion, Flow Then Structure
- **Relationship**: Strategy frameworks inform approach decisions

### Close (14 frameworks)
- **Tier 1**: Offer Structure, Crossroads, Future Pace, Guarantee, Scarcity, Price Minimization
- **Tier 2-3**: Bonus Strategy, Risk Transfer, FAQ Extension
- **Relationship**: Close frameworks are the culmination of all other work

### Stories (12 frameworks)
- **Tier 1**: Hero's Journey, Story Arc, Origin Story Formula
- **Tier 2-3**: Nightmare Story, Transformation Story, Conflict Injection
- **Relationship**: Story frameworks pair with leads and body copy

### Body Copy (10 frameworks)
- **Tier 1**: Dual Journey Structure, Energy Audit, People Doing Stuff
- **Tier 2-3**: Explainer Trap, No-Filler, Caveman Simple
- **Relationship**: Body frameworks apply to all copy sections

## Common Pairing Patterns

### 1. Strategy → All Other Components
```
FLOW: Strategy determines approach → All components execute it

Recommended sequence:
  strategy/4-cs-2-hs → (evaluate any copy)
  strategy/market-sophistication → (choose lead type)
  strategy/just-talk → (voice for all sections)
```

### 2. Psychology → Close
```
FLOW: Psychology identifies triggers → Close activates them for action

Recommended pairs:
  psychology/10-emotional-triggers
    ↔ close/future-pace
    ↔ close/crossroads-close
```

### 3. Headlines ↔ Bullets ↔ Leads
```
FLOW: Headlines grab attention → Bullets build curiosity → Leads maintain it

Recommended pairs:
  headlines/100-method
    ↔ bullets/bullet-polishing
    ↔ leads/dual-journey
```

### 4. Stories ↔ Body Copy
```
FLOW: Stories create engagement → Body copy maintains energy

Recommended pairs:
  stories/hero-journey
    ↔ body/people-doing-stuff
    ↔ body/energy-audit
```

### 5. Process → All Components
```
FLOW: Process informs HOW you work on any component

Universal application:
  process/volume-method → More ideas = better ideas
  process/make-it-better → Continuous improvement
  process/80-20-completion → Know when to stop
```

## Using Relationships for Workflow

### Complete Sales Letter Workflow
1. **Strategy**: 4 C's + 2 H's, Market Sophistication, Big Idea
2. **Psychology**: 10 Emotional Triggers, Fear vs Greed
3. **Headlines**: 100 Method, Opposites, Mechanisms
4. **Leads**: Dual Journey, Resistance-Free
5. **Stories**: Choose type (Origin, Nightmare, Transformation)
6. **Bullets**: Polishing Method, Specificity Stack
7. **Body**: Energy Audit, People Doing Stuff
8. **Close**: Future Pace, Crossroads, Offer Structure

### Quick Copy Review Workflow
1. **Strategy**: 4 C's check (Clarity, Compelling, Credible, Change)
2. **Strategy**: 2 H's check (Honest, Heart-to-Heart)
3. **Body**: Dual Journey check
4. **Body**: Energy Audit (find best paragraph, bring others up)

### Headline Session Workflow
1. **Strategy**: Big Idea Framework
2. **Strategy**: Market Sophistication (which stage?)
3. **Headlines**: 100 Method
4. **Headlines**: Opposite Technique
5. **Headlines**: Mechanism (if Stage 3+)

### The Deutsch Difference
What makes Deutsch methodology unique:
1. **Just Talk** - Access persuader brain, not writer brain
2. **Dual Journey** - Never separate logic from emotion
3. **4 C's + 2 H's** - Simple evaluation framework
4. **Caveman Simple** - Distracted readers need clarity
5. **Crossroads Close** - Life-changing decision, not purchase
