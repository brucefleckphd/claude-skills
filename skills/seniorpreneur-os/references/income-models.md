## Offer Evaluation Matrix

Use this to score any new offer before committing time:

| Criterion | Weight | Score (1–5) | Notes |
|-----------|--------|-------------|-------|
| Recurring revenue (vs. one-off) | 30% | | |
| Leverage (scales without proportional hours) | 25% | | |
| Aligns with writing/teaching/speaking | 20% | | |
| MRR impact (moves toward next milestone) | 15% | | |
| Fits current weekly time budget | 10% | | |

**Weighted score = sum of (weight × score)**

- 4.0–5.0: Proceed immediately
- 3.0–3.9: Proceed with constraints (e.g., delegate delivery)
- 2.0–2.9: Redesign the offer before committing
- Under 2.0: Decline or defer

## Pricing Philosophy

- **Underpricing is a lifestyle risk.** At 20–30 hrs/week, every client slot is scarce. Price to reflect that scarcity.
- **Annual plans reduce churn.** Offer annual membership at 2 months free (e.g., $997/year vs. $97/month).
- **Certification increases licensing value.** A certified graduate is a marketing asset for corporate buyers.
- **Raise prices as demand grows.** Use waitlists and sold-out cohorts as evidence to justify price increases.
