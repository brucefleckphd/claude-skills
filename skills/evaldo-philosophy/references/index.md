# Evaldo Philosophy Frameworks Index

Quick reference for all philosophy frameworks.

## Foundation (Core Methodology)

| Framework | Description | Use When |
|-----------|-------------|----------|
| philosophy-one-belief | The 16-word formula that controls all copy | Starting any project; ensuring focus |
| philosophy-croc-brain | Three filters your message must pass | Evaluating ideas; checking attention triggers |
| philosophy-three-box | Emotional + Simple + New test | Validating ideas before writing |

## Mindset (Copywriter Psychology)

| Framework | Description | Use When |
|-----------|-------------|----------|
| philosophy-domination-mindset | Compete to win, not just to play | Facing competition; setting goals |
| philosophy-13-things | What mentally strong copywriters don't do | Building resilience; avoiding common traps |
| philosophy-thoughts-feelings-desires | Getting inside the prospect's head | Planning copy; understanding audience |

## Structure (VSL Approach)

| Framework | Description | Use When |
|-----------|-------------|----------|
| philosophy-vsl-mini-movie | VSLs as entertainment, not ads | Planning video structure |
| philosophy-first-ask-last-ask | Strategy of asks throughout copy | Planning commitment sequence |

## Framework Relationships

```
philosophy-one-belief (start here)
    ↓
philosophy-three-box (validate idea)
    ↓
philosophy-croc-brain (check all filters)
    ↓
philosophy-thoughts-feelings-desires (understand prospect)
    ↓
philosophy-vsl-mini-movie (structure approach)
```

## Quick Filters

**Before writing anything, confirm:**
- [ ] One Belief articulated clearly
- [ ] Passes Three Box test (Emotional + Simple + New)
- [ ] Passes all three Croc Brain filters
- [ ] Know the prospect's thoughts, feelings, desires
