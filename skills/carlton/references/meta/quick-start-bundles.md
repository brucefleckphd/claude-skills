# Carlton Quick-Start Bundles

Pre-packaged framework sets for common copywriting tasks.

---

## Bundle 1: The Hook Bundle

**When to use:** Starting any new sales piece, need attention-grabber

**Skills to invoke:**
1. `carlton-hooks` - Develop incongruous juxtaposition
2. `carlton-headlines` - Translate hook to headline
3. `carlton-psychology` - Identify A-Brain triggers for hook

**Quick process:**
1. List all "weakness + amazing result" angles
2. Find strongest incongruous juxtaposition
3. Test against zombie state criteria
4. Write 5-10 headline variations
5. Select best with deck copy support

**Templates:**
- Weakness + Amazing Result
- Unlikely Source + Secret
- They Laughed... Until
- Hidden Knowledge

## Bundle 2: The Bullet Bundle

**When to use:** Need compelling bullets for sales page or email

**Skills to invoke:**
1. `carlton-features-benefits` - Build feature → benefit list
2. `carlton-bullets` - Apply one-two punch
3. `carlton-voice` - Ensure bar room feel

**Quick process:**
1. List all features
2. Apply "that means..." to each (go deep)
3. Write bullets using templates
4. Name techniques for perceived value
5. Add giveaway bullets for goodwill
6. Vary templates for roller coaster effect

**One-Two Punch reminder:**
- Punch One: Practical use OR positioning
- Punch Two: Shock/delight OR assault on common sense

## Bundle 3: The Close Bundle

**When to use:** Writing the final selling section

**Skills to invoke:**
1. `carlton-close` - 5-part structure
2. `carlton-turbulence` - Urgency elements
3. `carlton-offers` - Guarantee and risk reversal

**Quick process:**
1. Summarize offer ("Here's the deal")
2. List everything with values
3. Present price with contrast
4. State guarantee (benefit-rich, not risk-focused)
5. Walk through exact action steps
6. Add turbulence (urgency/takeaway)
7. Assume the sale throughout

**Close checklist:**
- [ ] No hedging ("maybe", "if you want")
- [ ] First person intimacy
- [ ] Walk-through instructions
- [ ] Assumes sale, doesn't beg

## Bundle 4: The Voice Bundle

**When to use:** Copy sounds too stiff/corporate, needs personality

**Skills to invoke:**
1. `carlton-voice` - Bar room conversation
2. `carlton-psychology` - A-Brain triggers
3. Voice patterns from `carlton/references/voice-patterns.md`

**Quick fixes:**
1. Read aloud - rewrite anything that sounds stiff
2. Replace "we" with "I" where appropriate
3. Add contractions ("you're" not "you are")
4. Inject specific details (not generalities)
5. Cut any throat-clearing opening
6. Add A-Brain trigger words
7. Include bar room expressions

**Bar room test:**
Would you say this to a friend in a bar and have them lean in?

## Bundle 5: The Research Bundle

**When to use:** Starting new project, don't know the market

**Skills to invoke:**
1. `carlton-research` - Market, avatar, USP
2. `carlton-features-benefits` - "That means" translation
3. `carlton-psychology` - A-Brain trigger mapping

**Quick process:**
1. Answer Hot Seat questions (Who? What? Why? For whom?)
2. Create specific avatar (demographics + psychographics)
3. Develop USP (We help [group] do [benefit] [better])
4. List features → apply "that means" → go deep
5. Identify A-Brain triggers for this market

**Avatar template:**
- Name: [give them a name]
- Pain: [what hurts?]
- Desire: [what do they want?]
- Fear: [what scares them?]
- Language: [how do they talk about this?]

## Bundle 6: The Full Letter Bundle

**When to use:** Writing complete sales letter from scratch

**Skills to invoke:**
1. `carlton` (orchestrator) - Full 17-point workflow
2. All element skills in sequence
3. `carlton-critic` (agent) for validation

**Full process:**
1. **Research (Steps 1-4)**
   - Market research
   - Avatar creation
   - USP development
   - Feature/benefit list

2. **Core Elements (Steps 5-7)**
   - Write bullets
   - Gather testimonials
   - Develop hook

3. **Salesmanship (Steps 8-10)**
   - Get into Go-To Guy mindset
   - Develop voice and story
   - Write headlines

4. **Structure (Steps 11-12)**
   - Plan funnel and delivery
   - Apply basic sales message formula

5. **Close (Steps 13-17)**
   - Make huge promise
   - Reverse risk
   - Create turbulence
   - Add bonuses
   - Write close

6. **Validate**
   - Run through `carlton-critic`
   - Fix issues
   - Final review

## Bundle 7: The Testimonial Bundle

**When to use:** Need to improve weak testimonials

**Skills to invoke:**
1. `carlton-testimonials` - Short, Specific, Spicy
2. `carlton-voice` - Authenticity check

**Quick process:**
1. Evaluate each: Short? Specific? Spicy?
2. Cut any rambling
3. Add specific numbers/results
4. Inject emotional punch
5. Vary voices (shouldn't all sound the same)
6. Ensure results-based, not feeling-based

**The ask:**
> "Dude, you owe me a testimonial."

**Prompt questions:**
- What was your situation BEFORE?
- What specific results did you get?
- How quickly did you see results?
